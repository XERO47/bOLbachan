📺 Use as Virtual Camera

The deepfake output is now captured as a MediaStream. To use it in video calls:

Option 1 — Share browser tab:
In Zoom/Meet/Teams, choose "Share Screen" → select this browser tab.

Option 2 — OBS Virtual Camera (best):

1. Open OBS → Add Source → Browser Source 2. URL: http://localhost:8000 3. Start Virtual Camera in OBS 4. In Zoom/Meet, select "OBS Virtual Camera"

Option 3 — Linux v4l2loopback:

ffmpeg -f x11grab -s 480x360 -r 25 \ -i :0.0+<x>,<y> \ -vf format=yuv420p \ -f v4l2 /dev/video10
